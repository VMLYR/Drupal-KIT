# Cookie compliance module

You will need to make changes to this component if you use EUCC instead of Dropsolid's cookie compliance module.

Be advised that at the time of writing this, the component in this styleguide (+ correcponding twig and Sass),
are for the Dropsolid Cookie Compliance module, NOT the EU Cookie Compliance module because the EUCC is still being worked on.
